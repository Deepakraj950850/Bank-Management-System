# Bank-Management-System
A Java-based banking application that allows users to manage accounts, deposits, withdrawals, and transaction

## 🚀 Features
- Create Account
- Deposit & Withdraw Money
- Balance Check
- Transaction History

## 🛠️ Technologies Used
- Core Java
- Java Swing
- OOPs Concepts

## ▶️ How to Run
1. Clone the repository
2. Open in Eclipse/IntelliJ
3. Run Main.java
